Fast answer: your Google Forms client intake template should feed answers into a Google Sheet, score each lead across six buying signals, and send only qualified prospects to your booking link.

How to structure the Google Form

✓ Section 1: Contact basics

Name, email, website/company, and role. Keep it simple so serious leads start quickly.

✓ Section 2: Project outcome

Ask what they want to launch, fix, improve, or avoid. This tells you whether the enquiry matches your offer.

✓ Section 3: Budget and timing

Use multiple-choice ranges for budget and deadline. Ranges are easier to score than open text.

✓ Section 4: Decision and scope

Ask who approves, who gives feedback, what assets exist, and what could make the project difficult.

Copy-ready Google Forms questions

  1. What is your name and email?
  2. What company, website, or project is this for?
  3. What result are you trying to create?
  4. Which service do you need help with?
  5. What have you already tried?
  6. What budget range is already approved?
  7. What deadline matters, and why?
  8. Who makes the final decision?
  9. Who else will review or give feedback?
  10. What assets, content, logins, or examples do you already have?
  11. What would make this project hard to complete?
  12. If we are a fit, are you ready to book a discovery call this week?

Tip: mark only the deal-breaker questions as required. If every field is required, good leads may abandon the form.

Google Sheets scoring formula

Send form responses to Google Sheets, then add a “Lead score” column. Start simple: give each signal 0, 1, or 2 points.

  • Budget: 2 if it meets your minimum, 1 if unclear, 0 if below minimum.
  • Deadline: 2 if realistic and urgent, 1 if flexible, 0 if panic or vague.
  • Authority: 2 if decision maker, 1 if influencer, 0 if unknown committee.
  • Service fit: 2 if exact offer match, 1 if adjacent, 0 if outside your work.
  • Scope clarity: 2 if outcome and constraints are clear, 1 if partial, 0 if vague.
  • Readiness: 2 if ready now, 1 if needs one clarification, 0 if just browsing.
