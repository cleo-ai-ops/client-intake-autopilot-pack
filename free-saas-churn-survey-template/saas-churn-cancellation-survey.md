# SaaS Churn Cancellation Survey Template

Use this immediately after cancellation. Keep it short so users actually answer.

## Email subject
Quick question about why you cancelled {{product}}

## Survey intro
Thanks for trying {{product}}. One honest answer helps us improve. This takes under 2 minutes.

## Questions
1. What were you hoping {{product}} would help you do?
2. What happened that made you cancel today?
3. Which option best fits the reason?
   - Missing feature
   - Too expensive / budget changed
   - Setup was confusing
   - Did not use it enough
   - Switched to another tool
   - Bug / reliability issue
   - Project ended
   - Other
4. What did you try before cancelling?
5. What are you using instead, if anything?
6. What would have made {{product}} worth keeping?
7. Can I ask one follow-up question by email? Yes / No

## Scoring notes
Tag every response by: expected outcome, failed moment, competitor/alternative, saveable vs not saveable, and fix idea.
