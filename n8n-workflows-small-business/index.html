10 n8n Workflows That Save Small Businesses 10+ Hours Per Week

n8n is the most powerful free workflow automation tool available — but most small businesses never get past "hello world" automations. These are 10 practical workflow patterns small businesses can adapt first. Each paid workflow is importable, configurable, and includes setup notes plus test data.

The 10 workflows

1. Lead Qualification Router

CRMSalesTime saved: ~3 hrs/week

When a new lead arrives (form submission, email, or CRM entry), automatically score them based on company size, budget range, project type, and timeline. High-scoring leads go to a priority Slack channel and get a fast follow-up email. Low-scoring leads get a polite nurture sequence.

Nodes: Webhook → Switch (score branches) → Slack / Email / CRM update

Works with: Any CRM (HubSpot, Pipedrive), Slack, Gmail

2. Invoice Reminder Sequence

FinanceCashflowTime saved: ~2 hrs/week

Check unpaid invoices daily. At 3 days overdue: send a friendly reminder. At 7 days: send a firmer follow-up with payment link. At 14 days: escalate to a phone call reminder and flag in your accounting system.

Nodes: Cron (daily) → HTTP (get invoices) → IF (overdue check) → Switch (days overdue) → Email / Slack

Works with: Stripe, Xero, QuickBooks, FreeAgent

3. Client Onboarding Automator

Client SuccessOnboardingTime saved: ~2 hrs/client

When a new client signs (triggered by Stripe payment or CRM stage change), automatically: create their project folder in Google Drive, send a welcome email with intake form, create a Slack channel, add them to your CRM, and schedule a kickoff meeting.

Nodes: Webhook/Stripe trigger → Google Drive → Gmail → Slack → CRM → Calendar

4. Support Ticket Triage

SupportCustomer SuccessTime saved: ~1.5 hrs/week

Route incoming support emails based on keywords and sentiment. Billing issues → finance Slack channel. Bug reports → engineering with error details. Feature requests → product board. General questions → auto-reply with FAQ link.

Nodes: Gmail trigger → Text Analysis (keyword matching) → Switch (category) → Slack / Trello / Auto-reply

5. Weekly Report Generator

ReportingAnalyticsTime saved: ~2 hrs/week

Every Monday at 8 AM, pull data from your key tools (revenue from Stripe, new leads from CRM, support tickets resolved, website traffic), compile into a summary email, and send to stakeholders.

Nodes: Cron → HTTP (multiple API calls) → Merge → HTML template → Email

6. Content Republisher

MarketingSocialTime saved: ~1.5 hrs/week

When you publish a new blog post, automatically create social media posts for Twitter, LinkedIn, and a newsletter draft. Each platform gets a tailored format — thread for Twitter, professional summary for LinkedIn, digest format for email.

Nodes: RSS/Webhook trigger → HTTP (AI formatting) → Twitter API / LinkedIn API / Gmail draft

7. Contract Expiry Alert

OperationsLegalTime saved: ~0.5 hrs/week

Monitor contract end dates and send renewal reminders 30, 14, and 7 days before expiry. Include the contract terms and suggested renewal options. Never miss a renewal again.

Nodes: Cron → Database lookup → IF (date proximity) → Email / Slack

8. Meeting Prep Briefing

ProductivityCalendarTime saved: ~1 hr/week

15 minutes before each meeting, pull the attendee's CRM data, recent email thread, project status, and open action items into a Slack DM briefing note.

Nodes: Cron (every 15 min) → Google Calendar → CRM lookup → Gmail thread → Slack

9. Expense Receipt Processor

FinanceAccountingTime saved: ~1 hr/week

Forward receipt emails to a dedicated address. The workflow extracts amount, vendor, date, and category, then creates an entry in your accounting software and files the receipt in Google Drive.

Nodes: Gmail trigger → Text extraction → Google Sheets / Accounting API → Google Drive

10. Testimonial Requester

MarketingSocial ProofTime saved: ~0.5 hrs/week

After a project milestone is completed or a support ticket is resolved with positive feedback, wait 3 days then send a testimonial request email. If they respond, log it in your testimonial database.

Nodes: CRM/Support trigger → Wait (3 days) → Email → IF (response) → Google Sheets / Slack

How to get started with n8n

If you're new to n8n:

  1. Self-host for free: Install n8n with Docker (docker run -p 5678:5678 n8nio/n8n) or use n8n Cloud's free tier for up to 5 workflows.
  2. Start with one workflow: Don't try to automate everything at once. Pick the workflow above that addresses your biggest time sink.
  3. Import and customize: Download the workflow JSON, import it into n8n, and update the credentials (API keys, email addresses, Slack channels).
  4. Test before going live: Use the test data included in the pack to verify each workflow behaves correctly with your setup.
  5. Add more workflows gradually: Once the first one runs reliably for a week, add the next one.
